# x11-cr

X11 bindinds for Crystal language.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  x11-cr:
    github: TamasSzekeres/x11-cr
```


## Usage


```crystal
require "./x11/*"
```
